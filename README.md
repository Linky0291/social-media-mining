## Introduction
This project is a high-performance web scraper designed to extract comment data from Douyin (TikTok China). It utilizes **Playwright** for dynamic content rendering and **Pandas** for data structuring.

## Key Features
- **Dynamic Rendering:** Handles infinite scrolling and dynamic AJAX loading.
- **Anti-Scraping Bypass:** Implements stealth techniques to mimic human behavior.
- **Data Pipeline:** Automatically cleans and exports data to CSV/JSON for sentiment analysis.

## Tech Stack
- Python 3.9+
- Playwright
- Pandas
- BeautifulSoup4
